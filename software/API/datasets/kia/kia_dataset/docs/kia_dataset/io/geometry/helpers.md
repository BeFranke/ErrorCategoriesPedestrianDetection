### *def* **magnitude_of_vec**(x)

Compute the magnitude of a 3 dimensional vector (array with 3 entries).

* **x**: The list representing a vector.
* **returns**: The magnitude of the vector (euclidean length).


---
### *def* **normalize_vec**(x)

Normalize a 3 dimensional vector (array with 3 entries) to unit length.

* **x**: The list representing a vector that should be normalized.
* **returns**: The normalized vector x.
